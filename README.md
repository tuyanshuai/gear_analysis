# 齿轮分析系统 (Gear Analysis System)

基于齿轮轮廓的分析软件，参考海克斯康Quindos功能界面设计，提供专业的齿轮参数计算和可视化功能。

## 功能特点

- ✅ **齿轮参数计算**：自动计算齿轮的核心参数
  - 分度圆直径
  - 齿顶圆直径
  - 齿根圆直径
  - 基圆直径
  - 跨棒距（M值）
  - 跨齿数
  - 测量球中心距
  - 压力角等

- ✅ **轮廓数据输入**：支持输入齿轮轮廓坐标点数据

- ✅ **可视化显示**：美观的齿轮图形绘制
  - 分度圆、齿顶圆、齿根圆、基圆显示
  - 齿轮齿形可视化
  - 轮廓点绘制
  - 可缩放、可拖动的交互式视图

- ✅ **专业界面**：参考Quindos测量软件的专业界面设计
  - 现代化的UI设计
  - 清晰的数据展示
  - 实时计算结果更新

## 使用方法

1. **打开应用**
   - 直接在浏览器中打开 `index.html` 文件

2. **输入基本参数**
   - 齿数 (Z)：齿轮的齿数
   - 模数 (m)：齿轮模数
   - 压力角 (α)：齿轮压力角，通常为20°
   - 齿顶高系数 (ha*)：通常为1
   - 齿根高系数 (c*)：通常为0.25
   - 测量球直径 (d)：用于跨棒距测量的球直径

3. **输入轮廓数据（可选）**
   - 在"轮廓数据"文本框中输入坐标点
   - 格式：每行一个点，格式为 `x, y` 或 `x y`
   - 例如：
     ```
     10.0, 5.0
     10.5, 5.2
     11.0, 5.5
     ```

4. **计算参数**
   - 点击"计算"按钮或修改参数后自动计算
   - 结果会显示在右侧面板
   - 齿轮图形会显示在中间可视化区域

5. **视图操作**
   - 鼠标滚轮：缩放视图
   - 鼠标拖动：平移视图
   - 视图控制按钮：放大、缩小、重置视图

## 计算公式

### 主要参数
- **分度圆直径**：d = m × Z
- **齿顶圆直径**：da = d + 2 × m × ha*
- **齿根圆直径**：df = d - 2 × m × (ha* + c*)
- **基圆直径**：db = d × cos(α)

### 跨棒距
跨棒距（M值）是齿轮测量中的重要参数，用于检验齿轮的精度。计算公式考虑了：
- 测量球直径
- 跨齿数
- 压力角
- 基圆直径

## 注意事项

1. 输入参数时请确保数值合理
2. 压力角通常在14.5°到25°之间
3. 轮廓数据应为有效的坐标点
4. 计算结果仅供参考，实际应用请结合专业测量
